# 项目简介

使用 Python 对糖尿病数据集进行线性回归建模，预测糖尿病患者的一年后疾病进展指标（target）。项目涵盖完整的机器学习工作流程，包括数据探索、预处理、模型训练、评估及可视化。

# 技术栈

Python 3.9

numpy, pandas, matplotlib, seaborn, scikit-learn

# 数据来源

数据集来自 sklearn.datasets.load_diabetes()

样本数：442

特征：10 个连续特征，包括 BMI、血压（bp）、血液指标（s1-s6）等

目标：target — 一年后糖尿病疾病进展指标

数据已标准化，无缺失值
